## 🎯 核心建议

### 前端开发遵循原则

```
优先使用 HDty 框架提供的组件
    ↓
必要时在框架组件基础上扩展
    ↓
万不得已才从头开发新组件
```

### 具体组件对应关系

| 功能需求         | HDty 框架组件        | 所在模块                             |
| ---------------- | -------------------- | ------------------------------------ |
| 动态表单渲染     | hd-form              | src/components/hd-form/              |
| 多级表头表格     | hd-build-table       | src/components/hd-build-table/       |
| 字典选择（单选） | hd-dict-select       | src/components/hd-dict-select/       |
| 字典选择（多选） | hd-dict-checkbox     | src/components/hd-dict-checkbox/     |
| 字典选择（级联） | hd-dict-cascader     | src/components/hd-dict-cascader/     |
| 日期选择         | hd-date-picker       | src/components/hd-date-picker/       |
| 日期范围         | hd-date-range-picker | src/components/hd-date-range-picker/ |
| 图片上传         | hd-image-upload      | src/components/hd-image-upload/      |
| 二维码生成       | hd-qrcode            | src/components/hd-qrcode/            |
| 视频播放         | hd-video-play        | src/components/hd-video-play/        |
| 权限控制         | hd-btn-authority     | src/components/hd-btn-authority/     |
| 本地存储         | storage 工具         | src/utils/storage.js                 |
| HTTP 请求        | axios-init + api     | src/config/axios-init.js + src/api/  |
