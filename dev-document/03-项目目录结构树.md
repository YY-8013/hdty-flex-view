## 快速查找指南

### 按功能查找
- **组件**: `src/components/` 或 `src/views/business/flex-view/components/`
- **API接口**: `src/api/` 或相应模块下的 `api/` 目录
- **工具函数**: `src/utils/`
- **路由配置**: `src/router/`
- **状态管理**: `src/store/`

### 按业务查找
- **数据晾晒**: `src/views/business/flex-view/`
- **用户管理**: `src/relies/user/`
- **字典管理**: `src/relies/dict/`
- **系统配置**: `src/relies/custom/`、`src/config/`

### 文档查找
- **开发计划**: `project-docs/02-前端开发计划.md`
- **组件规范**: `project-docs/05-HDty组件使用规范与模板.md`
- **架构设计**: `dev-md/系统架构设计文档.md`
- **工具说明**: `mdList/` 目录下相应文件

## 注意事项

1. **flex-view 模块**: 所有新增组件、工具方法必须统一存放于 `src/views/business/flex-view/` 目录下
2. **样式变量**: 避免使用未定义的 Sass 变量，优先使用 HDty 框架提供的样式
3. **运行环境**: Node.js 14.x + npm 6.x（推荐 Node 14.21.3）
4. **开发服务器**: `npm run dev` 启动，监听 `0.0.0.0:8060`
5. **文档维护**: 每次有重要目录变化时，应更新本文档
