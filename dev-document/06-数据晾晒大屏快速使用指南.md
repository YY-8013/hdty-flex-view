## 🎯 功能概览

数据晾晒大屏统计展示是一个**可视化统计数据展示平台**，具有以下特色：

- ✅ **双主题切换**: 警用蓝 / 常规主题，一键切换
- ✅ **动态列配置**: 从后台配置读取，无需修改代码
- ✅ **Mock数据**: 鄂尔多斯市9个旗区模拟数据
- ✅ **交互友好**: 点击查看明细，分页查询便捷

## 🎨 主题切换

### 警用蓝主题
**适用场景**: 警务系统、严肃场合

**特点**:
- 🔵 深蓝色渐变背景
- 💪 稳重、专业、权威
- 👁️ 高对比度，易读性强

**效果预览**:
```
顶部: 深蓝渐变 (#1e3a8a → #2563eb)
表头: 蓝色渐变 (#3b82f6 → #2563eb)
```

### 常规主题
**适用场景**: 日常办公、常规展示

**特点**:
- 🟣 紫色渐变背景
- 🌈 柔和、舒适、现代
- 😊 视觉友好，长时间观看不疲劳

**效果预览**:
```
顶部: 紫色渐变 (#667eea → #764ba2)
表头: 紫色渐变 (#a78bfa → #8b5cf6)
```

### 如何切换？
1. 点击右上角「🎨 切换主题」
2. 选择「警用蓝主题」或「常规主题」
3. 系统自动保存，下次访问自动恢复

## 👆 交互操作

### 查看明细数据

**操作步骤**:
1. 找到带下划线的数值单元格
2. 点击该单元格
3. 弹出明细数据窗口
4. 查看/编辑/删除明细数据

**示例**:
```
点击「东胜区」的「总数: 3618」
→ 弹出「实有人口总数明细」窗口
→ 显示该机构的所有实有人口明细记录
```

### 分页查询

**操作**:
- 点击页码切换页面
- 修改每页条数（20/50/100/200）
- 使用「上一页」「下一页」按钮

**操作**:
1. 点击「导出」按钮
2. 选择导出格式（Excel）
3. 下载统计数据

## ⚙️ 配置说明

### 列配置管理

**路径**: 配置管理 → 列配置管理

**如何添加新列**:
1. 进入列配置管理页面
2. 点击「新增」按钮
3. 填写列信息：
   - 列名称: 如「总数」
   - 属性名: 如「syrk_total」
   - 关联表单: 选择明细表单（可选）
   - 启用状态: 启用
4. 保存配置
5. 刷新统计页面，新列自动显示

### 表单配置管理

**路径**: 配置管理 → 表单配置管理

**如何配置明细表单**:
1. 进入表单配置管理页面
2. 创建新表单或编辑现有表单
3. 配置表单字段（新增/编辑/删除）
4. 在列配置中关联该表单
5. 点击统计列时，自动弹出该表单

## 🐛 常见问题

### Q1: 页面打不开？
**A**: 
1. 检查网络连接
2. 清除浏览器缓存
3. 联系管理员检查权限

### Q2: 主题切换后没有生效？
**A**: 
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 检查localStorage是否被禁用

### Q3: 点击单元格没有反应？
**A**: 
1. 确认该列是否配置了关联表单
2. 只有带下划线的单元格可以点击
3. 检查浏览器控制台是否有错误

### Q4: 数据不准确？
**A**: 
1. 当前使用Mock数据（模拟数据）
2. 正式环境需对接真实API
3. 联系技术团队对接数据接口

### Q5: 如何修改显示的列？
**A**: 
1. 进入「列配置管理」模块
2. 编辑/添加列配置
3. 设置启用状态和注销状态
4. 刷新统计页面查看效果
